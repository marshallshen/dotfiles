## Marshall Shen's Dot Files

These are config files to bootstrap a system with tools I need.

### Installation

#### Mac OS X
git clone git@github.com:marshallshen/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
Go to `mac/` folder
Run the following command on the machine:

```
rake install
```

#### Linux
git clone git@github.com:marshallshen/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
Go to `linux/` folder
